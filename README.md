#  Jarvis Voice Assistant

A Python-based voice assistant that can:

- Respond to voice commands
- Tell time and date
- Search Wikipedia
- Open Google and YouTube
- Play music
- Voice interaction using Speech Recognition and Text-to-Speech

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- Wikipedia API

## How to Run

pip install SpeechRecognition pyttsx3 wikipedia

python health_assistant.py
